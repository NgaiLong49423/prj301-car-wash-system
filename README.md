## 🚀 Cách Chạy Dự Án

### Trong NetBeans:
1. Mở NetBeans
2. **File** → **Open Project** → Chọn thư mục dự án
3. Chọn **Run Project** hoặc nhấn `F6`
4. Trình duyệt sẽ mở tự động tại `http://localhost:8080/[project-name]`

### Sử dụng Terminal:
```bash
# Build project
cd d:\Semester 4\PRJ301\PRJ301
mvn clean build

# Chạy trên Tomcat
catalina run
```
