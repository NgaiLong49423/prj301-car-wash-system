## ⚠️ Xác Nhận Tuân Thủ Yêu Cầu

Chúng tôi xác nhận đã tuân thủ nghiêm ngặt các quy tắc và ràng buộc sau:

1. **Chỉ thay đổi Status trên Project Board**: Mọi thay đổi đều được thực hiện qua GitHub CLI nhắm vào field `Status`.
2. **Không sửa đổi source code**: Tuyệt đối không sửa đổi bất kỳ file mã nguồn nào của dự án.
3. **Không commit**: Không tạo bất kỳ commit nào lên Git repository của dự án.
4. **Không tạo branch**: Không tạo branch mới nào cho tác vụ này.
5. **Không đóng issue**: Không thực hiện đóng bất kỳ issue nào.
6. **Không đổi thông tin quản lý của issue**: Không đổi title, body, labels, milestone, hay assignee của các issue.
